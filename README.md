## Get Started

```sh
  git submodule add git@github.com:nozomiishii/.vscode.git
```

## Git Submodules

追加

```sh
  git submodule add <submoduleのgit_URL>　
```

更新

```sh
  git submodule foreach git pull
  ## or
  git submodule <app_name> git pull
  ## or
  git submodule update --recursive --remote
```
